Louis-Etienne Messier

The main goal of the internship was to use different training techniques : LoRA of different ranks and finetuning a ControlNet layer, in order to push a diffusion model to generate better metallic properties.

Goal #1

A whole new synthetic dataset was generated using 20 3D free car models found online, Blender, and The Laval Indoor HDR Dataset. The dataset was separated like this :

  • Training set : 1980 backgrounds, associated with one of the 16 car models
  • Evaluation set : 110 backgrounds, associated with one of the 2 car models
  • Test set : 110 backgrounds, associated with one of the 2 car models

The dataset was then used to train a rank 1 and rank 4 LoRA on the diffusion model, and also finetune the ControlNet unit. For each instance, the training process lasted 24h on two A100.

Goal #2

The secondary goal of the internship, was to explore the Infinigen library to generate procedural 3D rooms. This could potentially allow for an unlimited amount of pure and realistic synthetic data for Computer Vision tasks!

Details #2

Some results generated by my home computer with no GPU can be seen in the Results #2 tab, but those are simplified room layouts because of the unavailability of a GPU.

I tried to make the library run on the Compute Canada cluster system, with no success. I reached out to the Compute Canada staff multiple times, but the environment restrictions made it impossible to run Infinigen or debug it.

On the other hand, I managed to set up an Infinigen environment on one of JF's machine (Rachmaninoff). You just need to install conda locally and then follow the detailed procedure in the Github repo. You can then launch any of the commands described here to make it work!
